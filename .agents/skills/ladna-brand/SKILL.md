# Ladna Brand

## Workflow

Before designing or changing Ladna brand-facing UI, visuals, landing pages, or mascot variants, read `references/brandbook.md`.

Treat the saved mascot image as a concept reference only, not as a production image to place directly into pages:

- `public/assets/brand/mascot/concepts/ladna-mascot-concept-02-sporty-skirt.png`

Use the transparent production cutout when a page needs the current mascot directly:

- `public/assets/brand/mascot/ladna-mascot-sporty-cutout.png`
- `public/assets/brand/landing/ladna-landing-mascot-cutout.png`

Use the bundled skill copy only when creating derived visuals, prompts, or style references:

- `assets/mascot/concepts/ladna-mascot-concept-02-sporty-skirt.png`
- `assets/mascot/ladna-mascot-sporty-cutout.png`

Keep the brand direction sporty, warm, and SaaS-polished. Ladna is for sports studios and their clients, not only for back-office administrators.

## Guardrails

- Prefer the existing Ladna plum, lavender, cream, and ink palette over new dominant colors.
- Generate new fit-for-purpose mascot illustrations from the concept's image direction; do not crop or reuse the saved concept PNG as final landing art.
- Keep mascot characters separated from backgrounds. For landing heroes, use a transparent cutout over a page-level background instead of placing a rectangular illustration panel.
- Keep mascot usage adult, brand-safe, athletic-casual, and non-pin-up.
- Avoid school uniform cues, chibi proportions, office-only styling, neon palettes, and generic stock SaaS illustrations.
- For landing work, make Ladna itself and the sports-studio context visible in the first viewport.
- Reuse `public/brand/ladna-mark.svg` for the product mark unless the user explicitly asks for a logo redesign.
